# ronin-new-script 1 "May 2022" Ronin "User Manuals"

## SYNOPSIS

`ronin new script` [*options*] *PATH*

## DESCRIPTION

Creates a new standalone Ruby script.

## ARGUMENTS

*PATH*
	The path to the new script file.

## OPTIONS

`-h`, `--help`
  Print help information
