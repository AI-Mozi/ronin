# ronin-new-project 1 "May 2022" Ronin "User Manuals"

## SYNOPSIS

`ronin new project` [*options*] *PATH*

## DESCRIPTION

Creates a new project directory.

## ARGUMENTS

*PATH*
	The path to the new project directory.

## OPTIONS

`--git`
  Initializes a git repo.

`--rakefile`
  Creates a `Rakefile`.

`-h`, `--help`
  Print help information
